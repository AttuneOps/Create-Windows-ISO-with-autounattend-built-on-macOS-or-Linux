Step 1 - Confirm winpe_win2019_plain_bios.iso Does Not Exist

Checks if the WinPE plain BIOS ISO {automationWorkerBaseDirectory}/winpe_win2019_plain_bios.iso exists.

If it does, prints a message to ask user to confirm to delete it if they want to recreate it.

Only applies to BIOS kickstarts so if any of these parameters are set the string 'true':

  1. isWin10Bios
  2. isWinServerBios

Connect via ssh:

ssh {automationworkerlinuxuser}@{automationworkerlinuxnode}

Execute the following script: 

                
if {isWin10Bios} || {isWinServerBios}
then

    DIRECTORY="{automationWorkerBaseDirectory}"
    cd "${DIRECTORY}"
    
    ISO="winpe_win2019_plain_bios.iso"
    if [[ -f ${ISO} ]]
    then
        echo "Please delete ${DIRECTORY|/${ISO} to confirm you want to try again."
        exit 1
    fi
    
else
    echo "Skipping for UEFI ISO creation."
fi

Step 3 - Create WinPE Clean Build Files

Deletes the temporary build directory.

Step 3.1 - LIN Unmount Windows ISO

Unmounts {automationWorkerBaseDirectory}/build-{newOsNode.fqn}/winimg.

Connect via ssh:

ssh {automationworkerlinuxuserroot}@{automationworkerlinuxnode}

Execute the following script: 

                
MOUNT_FOLDER="winimg"
BUILD_DIRECTORY="{automationWorkerBaseDirectory}/build-{newOsNode.fqn}"
MOUNT_PATH="${BUILD_DIRECTORY}/${MOUNT_FOLDER}"

if df -h | grep --ignore-case \
    --quiet \
    "${MOUNT_PATH}"
then
    cd "${BUILD_DIRECTORY}"

    umount --verbose "${MOUNT_FOLDER}"
else
    echo "${MOUNT_PATH} is not mounted. Skipping."
fi

Step 3.2 - LIN Delete winimg Folder

Deletes the {automationWorkerBaseDirectory}/build-{newOsNode.fqn}/winimg folder.

Connect via ssh:

ssh {automationworkerlinuxuser}@{automationworkerlinuxnode}

Execute the following script: 

                
BUILD_DIRECTORY="{automationWorkerBaseDirectory}/build-{newOsNode.fqn}"
if [ ! -d ${BUILD_DIRECTORY} ]
then
    echo "Directory ${BUILD_DIRECTORY} does not exist. Skipping."
else
    cd "${BUILD_DIRECTORY}"
        
    DIR="winimg"
    [[ -d ${DIR} ]] && rm -rf ${DIR} && echo "Removed ${DIR}" || \
        echo "No directory to remove."
fi
